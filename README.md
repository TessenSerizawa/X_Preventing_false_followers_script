# X_Preventing_false_followers_script
## Overview
X(旧Twitter)にてフォロー/フォロー解除ボタンの誤クリックによる誤フォロー/フォロー解除を防ぐTempermonkey用スクリプトです。
## Install
(0.導入していない人はTempermonkeyをインストールする)
1.このリポジトリをクローンまたはZIPダウンロード
2.Tenpermonkeyのユーティリティから「ファイルからインポート」、解凍したファイル内の X_Preventing_false_followers.js を選択してインストールする
## Note
* ブラウザの拡張機能アイコンをクリックし、Tempermonkeyを選択、スイッチボタンで簡単にオンオフできます。
